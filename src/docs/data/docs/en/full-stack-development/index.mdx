Welcome to the **Full Stack Web Development** course — your complete path to building modern web applications from frontend to backend.

<Aside variant="info" title="Course Format">
  Self-paced · 4 modules · Estimated 60 hours · Hands-on projects and real-world scenarios included.
</Aside>

---

## What You'll Learn

<Steps>
1. **Frontend Basics** — HTML, CSS, JavaScript, responsive design, and modern frameworks
2. **Backend Development** — Server-side programming, REST APIs, and authentication
3. **Databases & APIs** — SQL, NoSQL, ORMs, API design, and GraphQL
4. **Deployment & DevOps** — Hosting, CI/CD, monitoring, and production best practices
</Steps>

---

## Course Modules

| Module | Topics Covered |
|--------|---------------|
| **1. Frontend Basics** | HTML5, CSS3, JavaScript ES6+, React, responsive design, accessibility |
| **2. Backend Development** | Node.js, Express, Python, authentication, authorization, middleware |
| **3. Databases & APIs** | PostgreSQL, MongoDB, RESTful APIs, GraphQL, WebSockets |
| **4. Deployment & DevOps** | Docker, cloud hosting, CI/CD pipelines, monitoring, security |

---

## Who This Course Is For

- Aspiring full stack developers looking to build end-to-end applications
- Frontend developers wanting to expand into backend development
- Backend developers wanting to strengthen their frontend skills
- Entrepreneurs wanting to build and launch their own web products

---

## Prerequisites

<Aside variant="tip" title="No prior web development experience needed">
  Basic computer literacy and familiarity with using a code editor is helpful. No prior programming experience is required.
</Aside>
