MongoDB is the most widely used NoSQL database — it stores data as flexible JSON-like documents rather than rigid tables. This module teaches you everything from basic CRUD to advanced aggregation and production operations.

## Module Pages

| Page | What You'll Learn |
|------|-------------------|
| [MongoDB Fundamentals](/learn/full-stack-development/mongodb-databases/mongodb-fundamentals/) | Document model, collections, databases, BSON vs JSON, installing MongoDB, MongoDB Shell |

| [Query Operators & Filtering](/learn/full-stack-development/mongodb-databases/query-operators/) | Comparison (`$gt`, `$lt`, `$in`), logical (`$and`, `$or`), element (`$exists`), array (`$all`, `$elemMatch`) |
| [Indexing & Performance](/learn/full-stack-development/mongodb-databases/indexing/) | Single field, compound, multikey indexes, `explain()`, index selection, `sort` with indexes |
| [Aggregation Pipeline](/learn/full-stack-development/mongodb-databases/aggregation/) | `$match`, `$group`, `$sort`, `$project`, `$lookup`, `$unwind`, pipeline optimization |
| [Mongoose ODM](/learn/full-stack-development/mongodb-databases/mongoose/) | Schemas, models, validation, middleware (pre/post hooks), virtuals, populate (references) |

## Key Takeaways

By the end of this module, you'll be able to design MongoDB schemas, perform efficient queries using indexes and the aggregation pipeline, integrate MongoDB with Node.js using Mongoose, and deploy to MongoDB Atlas.
