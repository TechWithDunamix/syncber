JavaScript is the programming language of the web. Every modern website uses JavaScript to add interactivity, fetch data, and create dynamic user experiences. This module teaches you the language fundamentals.

## Module Pages

| Page | What You'll Learn |
|------|-------------------|
| [Variables & Data Types](/learn/full-stack-development/javascript-basics/variables-datatypes/) | `let`, `const`, `var`, strings, numbers, booleans, null, undefined, type coercion |

| [Control Flow](/learn/full-stack-development/javascript-basics/control-flow/) | `if`/`else`, `switch`, truthy/falsy values, short-circuit evaluation |
| [Loops & Iteration](/learn/full-stack-development/javascript-basics/loops/) | `for`, `while`, `do...while`, `for...of`, `for...in`, break and continue |
| [Functions](/learn/full-stack-development/javascript-basics/functions/) | Function declarations, expressions, arrow functions, parameters, return values, scope |
| [Objects](/learn/full-stack-development/javascript-basics/objects/) | Object literals, properties, methods, `this` keyword, object destructuring, spread operator |
| [Arrays](/learn/full-stack-development/javascript-basics/arrays/) | Array methods (push, pop, map, filter, reduce), spread, destructuring, multidimensional arrays |

## Key Takeaways

By the end of this module, you'll understand the core JavaScript language — variables, functions, objects, arrays, and control flow — enough to write simple programs and confidently move to advanced topics.
