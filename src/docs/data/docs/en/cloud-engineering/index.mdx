Welcome to the **Cloud Engineering** course — your complete path to designing and managing cloud infrastructure on AWS, Azure, and GCP.

<Aside variant="info" title="Course Format">
  Self-paced · 4 modules · Estimated 55 hours · Hands-on labs and real-world architecture projects included.
</Aside>

---

## What You'll Learn

<Steps>
1. **Cloud Fundamentals** — Cloud computing models, virtualization, networking, and security basics
2. **AWS Services** — EC2, S3, Lambda, VPC, RDS, IAM, and core AWS services
3. **Azure & GCP** — Azure VMs, Blob Storage, Functions, GCP Compute Engine, Cloud Storage
4. **Cloud Architecture** — High availability, cost optimization, security, and migration strategies
</Steps>

---

## Course Modules

| Module | Topics Covered |
|--------|---------------|
| **1. Cloud Fundamentals** | IaaS, PaaS, SaaS, virtualization, VPCs, subnets, cloud security models |
| **2. AWS Services** | EC2, S3, Lambda, API Gateway, RDS, DynamoDB, IAM, CloudFormation |
| **3. Azure & GCP** | Azure VMs, Blob Storage, Functions, GCP Compute, Cloud Storage, BigQuery |
| **4. Cloud Architecture** | HA/DR design, cost optimization, Well-Architected Framework, migration |

---

## Who This Course Is For

- IT professionals transitioning to cloud roles
- Developers wanting to build cloud-native applications
- System administrators moving from on-premise to cloud
- Solutions architects designing cloud infrastructure

## Prerequisites
