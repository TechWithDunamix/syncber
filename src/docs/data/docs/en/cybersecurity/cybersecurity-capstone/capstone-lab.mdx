## Objective

Apply skills from all 12 modules in an integrated lab: scan for vulnerabilities, deploy endpoint protection, configure MFA, and build a detection dashboard.

## Lab Exercises

### Exercise 1: Vulnerability Scan
Run an authenticated vulnerability scan against a test system. Use Nessus, OpenVAS, or a cloud-based scanner.

### Exercise 2: EDR Deployment
Deploy an EDR agent (Wazuh or Defender for Endpoint) on a test VM and verify telemetry is flowing.

### Exercise 3: MFA Policy
Configure a Conditional Access policy in Azure AD requiring MFA for all admin accounts.

### Exercise 4: SIEM Dashboard
Build a security dashboard in your SIEM showing:
- Failed logon attempts over time
- Top source IPs for authentication failures
- MFA failure rate
- EDR alert count by severity

## Deliverables

1. Vulnerability scan report with at least 10 findings
2. Screenshot of EDR dashboard showing active agent
3. Screenshot of MFA policy configuration
4. Screenshot of SIEM dashboard with security metrics
5. One-page executive summary of findings and recommendations
