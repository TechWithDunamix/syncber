The Cybersecurity Capstone module integrates everything you have learned across all 11 modules into a comprehensive security assessment project. You will evaluate a real-world organisation's security posture, identify gaps, and produce an actionable security improvement plan.

## Module Pages

| Page | Description |
|------|-------------|
| **Knowledge Map** | Complete cybersecurity knowledge map — all 12 modules, what you learned, how they connect |
| **Capstone Project** | Comprehensive security assessment — evaluate an organisation, produce a CISO-level report |
| **Capstone Lab** | Hands-on: security assessment of a sample organisation, vulnerability scan, control mapping |
| **Career Paths** | Cybersecurity certifications, job roles, salary ranges, and career progression |
| **Flashcards** | Test your knowledge |
