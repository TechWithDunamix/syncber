Welcome to the **Cyber Security and SOC Analyst** course — your complete path to understanding and defending against modern cyber threats while mastering the art of Security Operations Center analysis.

<Aside variant="info" title="Course Format">
  Self-paced · 13 modules · 5 phases · Hands-on labs, simulations, and real-world case studies included.
</Aside>

---

## What You'll Learn

<Steps>
1. **Phase 1: Security Foundations** — CIA triad, cryptography, network security, application security
2. **Phase 2: Threat Management** — SIEM, threat intelligence, incident response, forensics, vulnerability management
3. **Phase 3: Defense & Tools** — IAM, Zero Trust, pentesting tools, cloud and container security
4. **Phase 4: Governance & Operations** — Compliance frameworks, auditing, DevSecOps, security automation
5. **Phase 5: SOC Analyst** — SOC career paths, tiered analyst roles, SOC tooling, alert triage, playbooks, hands-on SOC lab
</Steps>

---

## Course Structure

| Phase | Modules | Topics Covered |
|-------|---------|---------------|
| **1. Security Foundations** | Security Fundamentals, Network Security, Application Security | CIA triad, risk management, cryptography, PKI, firewalls, IDS/IPS, VPNs, OWASP Top 10, secure coding, web security |
| **2. Threat Management** | Threat Detection, Incident Response, Vulnerability Management | SIEM, SOC, threat intelligence, MITRE ATT&CK, IR lifecycle, forensics, malware analysis, pentesting, bug bounty |
| **3. Defense & Tools** | IAM, Security Tools, Cloud Security | MFA, SSO, Zero Trust, PAM, Wireshark, Metasploit, Burp Suite, CSPM, CASB, container security |
| **4. Governance & Operations** | Compliance & Auditing, Security Automation, Capstone Project | ISO 27001, SOC 2, PCI DSS, SOAR, DevSecOps, SAST/DAST, policy as code, comprehensive security program |
| **5. SOC Analyst** | SOC Career Path, Tier 1-3 Roles, SOC Tooling, Alert Triage, Playbooks, SOC Lab | SOC tiers, analyst career progression, SIEM/EDR/SOAR mastery, alert triage methodology, playbook development, hands-on SOC simulation |
