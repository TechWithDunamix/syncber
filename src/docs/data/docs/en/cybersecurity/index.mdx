Welcome to the **Cybersecurity** course — your complete path to understanding and defending against modern cyber threats.

<Aside variant="info" title="Course Format">
  Self-paced · 4 modules · Estimated 45 hours · Hands-on labs, simulations, and real-world case studies included.
</Aside>

---

## What You'll Learn

<Steps>
1. **Security Fundamentals** — CIA triad, risk management, cryptography, and security frameworks
2. **Threat Detection & Response** — Monitoring, SIEM, incident response, and threat intelligence
3. **Network & Application Security** — Firewalls, IDS/IPS, secure coding, and vulnerability management
4. **Security Tools & Practices** — Penetration testing, compliance auditing, and security automation
</Steps>

---

## Course Modules

| Module | Topics Covered |
|--------|---------------|
| **1. Security Fundamentals** | CIA triad, risk assessment, encryption, PKI, NIST/CIS frameworks |
| **2. Threat Detection & Response** | SIEM, SOC operations, incident response lifecycle, forensics |
| **3. Network & Application Security** | Firewalls, IDS/IPS, WAF, OWASP Top 10, secure SDLC |
| **4. Security Tools & Practices** | Metasploit, Wireshark, Nessus, compliance audits, automation |

---

## Who This Course Is For

- IT professionals moving into security roles
- System administrators wanting to secure their infrastructure
- Developers wanting to build secure applications
- Anyone interested in a career in cybersecurity
